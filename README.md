# UniCloak Contracts

ZK privacy vault smart contracts built on Uniswap v4. Deposit ERC-20 tokens privately using zero-knowledge proofs (Groth16 / Circom), withdraw to any address without linking sender and recipient.

## Contracts

| Contract | Description |
|---|---|
| `UniCloakVault` | Private vault — ZK deposit/withdraw with Poseidon commitments |
| `UniCloakSafe` | Safe vault — password-derived guardian, no ZK required |
| `Groth16Verifier` | On-chain Groth16 proof verifier (generated by snarkjs) |

## How It Works

1. **Deposit** — user deposits tokens + a Poseidon commitment hash on-chain
2. **Prove** — user generates a ZK proof off-chain (Circom circuit) proving knowledge of the secret
3. **Withdraw** — relayer submits proof on-chain; contract verifies and sends tokens to any recipient

The nullifier prevents double-spending. The recipient address is part of the proof, preventing front-running.
